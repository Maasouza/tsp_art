### 概要 ###
画像操作ライブラリPillowを用いて、TSP Artと呼ばれる作品を生成するソースコードです。
TSPとはTraveling Salesman Problem(巡回セールスマン問題）の頭字語で、
オペレーションズ・リサーチや理論計算機科学の分野で長きに渡って研究されてきた問題です。
本ソースコードには、画像読込・点画・TSP解法・描画まで、TSP Art作成に必要な全項目が
含まれています。

### 動作確認 ###
OS : Windows 10
Python : 3.4
(Linux, UNIX, Macなどでも動作しますが、動作検証は上記でのみ実施）

### アーカイブの内容（フォルダ構成にて） ###
* srcフォルダ
    - tsp.py
    - tsp_art.py
    - stipple.py
* imgフォルダ
    - sample.jpg
        - サンプル用画像
    - result.jpg
        - サンプル画像をプログラムにかけた結果
* readme.txt
    - 本ファイル
